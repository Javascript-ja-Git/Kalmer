# Kolmanda loengu meeldetuletus

- tsüklid e korduvtegevus. 
- mitu korda tsüklit korratakse
  - `for`, `while`, `do while`
  - for on parem, kui teised e tuleb anda ette lenduri algväärtus, mis tingimusel lõppeb ja kuidas loendur iga tsükliga muutub.
  - for ja while vahe - while pisut lihtsam kirjutada, kuid vaja teatud asjad kindlasti kirja panna. 
  - do while, tehakse tegevus ennem ära, siis kontrollitakse

- funktsioonid e koodiblokk mida saab programmi teistes osades ka kasutada.
- F koosneb,  nimi  - function divide() {} esialgu koostatakse funktsioon, siis lisatakse käivitus , nt divide(4,2) konkreetsete numbritega. hetkel veel väärtust näha pole, tuleb lisada const result 

- Andmetüübid (primitiivsed)
    string, null, boolean, undefined jne
- tingimuslaused
    - if, if else, else if
